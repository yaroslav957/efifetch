# efifetch
![Stars](https://img.shields.io/github/stars/yaroslav957/efifetch?style=social)
![Issues](https://img.shields.io/github/issues/yaroslav957/efifetch?style=social)
![Preview](https://github.com/yaroslav957/efifetch/blob/main/assets/preview.png)

A simple system information tool for UEFI environments, designed to run directly from a UEFI shell. `efifetch` aims to provide concise and useful system details in a format inspired by popular system fetch tools like `neofetch` and `screenfetch`, but tailored for the unique UEFI runtime environment.
